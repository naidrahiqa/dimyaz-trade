## üèÅ Getting Started

Clone the repository and start trading in seconds.

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# 1. Clone the repo
git clone https://github.com/naidrahiqa/dimyaz-trade.git

# 2. Enter directory
cd dimyaz-trade

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
